fix_remote_address()

fix_remote_address(\Automattic\VIP\Proxy\(string)  $user_ip, \Automattic\VIP\Proxy\(string)  $remote_proxy_ip, \Automattic\VIP\Proxy\(string|\Automattic\VIP\Proxy\array)  $proxy_ip_whitelist) : \Automattic\VIP\Proxy\(bool)

Set REMOTE_ADDR to the end-user's IP address.

This allows more securely forwarding the origin IP address when your site is fronted by a proxy like Cloudflare or Akamai.

Without this, the Application will see the Remote Proxy's IP address as the REMOTE_ADDR. With this, if Remote Proxy's IP address matches a known whitelist, the Application will see the User's real IP address as REMOTE_ADDR.

Parameters

\Automattic\VIP\Proxy\(string) $user_ip

IP Address of the end-user passed through by the proxy.
\Automattic\VIP\Proxy\(string) $remote_proxy_ip

IP Address of the remote proxy.
\Automattic\VIP\Proxy\(string|\Automattic\VIP\Proxy\array) $proxy_ip_whitelist

Whitelisted IP addresses for the remote proxy. Supports IPv4 and IPv6, including CIDR format.

Returns

\Automattic\VIP\Proxy\(bool) —

true, if REMOTE_ADDR updated; false, if not.

fix_remote_address_from_ip_trail()

fix_remote_address_from_ip_trail(\Automattic\VIP\Proxy\(string)  $ip_trail, \Automattic\VIP\Proxy\(string|\Automattic\VIP\Proxy\array)  $proxy_ip_whitelist) : \Automattic\VIP\Proxy\(bool)

Set REMOTE_ADDR to the end-user's IP address from a trail of IP Addresses.

This allows more securely forwarding the origin IP address when there are multiple proxies in play.

Example setup: User => Remote Proxy (e.g. Cloudflare) => Local Proxy (Varnish) => Application (PHP/WP)

Without this, the Application will see the Remote Proxy's IP address as the REMOTE_ADDR. With this, if Remote Proxy's IP address matches a known whitelist, the Application will see the User's real IP address as REMOTE_ADDR.

Only two levels of proxies are supported.

Parameters

\Automattic\VIP\Proxy\(string) $ip_trail

Comma-separated list of IPs (something like user_ip, proxy_ip)
\Automattic\VIP\Proxy\(string|\Automattic\VIP\Proxy\array) $proxy_ip_whitelist

Whitelisted IP addresses for the remote proxy. Supports IPv4 and IPv6, including CIDR format.

Returns

\Automattic\VIP\Proxy\(bool) —

true, if REMOTE_ADDR updated; false, if not.
