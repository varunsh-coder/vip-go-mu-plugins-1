get_ip_addresses_from_ip_trail()

get_ip_addresses_from_ip_trail(string  $ip_trail) : array|boolean

Get a list of validated IP addresses from a comma-separated string expected to be passed as the X-IP-Trail HTTP request header.

Takes IP v4 or v6.

Only two levels of proxies are supported; i.e. fails if there's more than two IP addresses.

Fails if any IP address in the list is invalid.

Also checks the X-Forwarded-For header makes sense.
